# Santiago Benavides, Biomedical Engineering Portfolio

**Live site: https://alma26x.github.io/**

A portfolio of independent biomedical-engineering and neuroscience work, with a
current focus on HD surface-EMG decoding for prosthetic control.

## Sections
- **Brain Imaging & Neural Decoding:** surface-EMG gesture decoder (with live real-time demo) and a paper-format mini-review of the HD-sEMG generalization gap.
- **Medical Devices & Clinical Models:** a BCI concept render, a cardiac CT reconstruction, an HD-EMG bionic-arm concept, a parametric prosthetic finger in Fusion, and an ECG circuit.
- **Scaffold Design:** porous tissue scaffolds (cubic, sphere, vascular).
- **Protein Structure & AI Design:** structural visualization and AI-assisted protein design.
- **3D Animation & Visualization:** motor-unit and HD-EMG animations, a neuromuscular-junction render, and molecular visualizations of nAChR and NMDA receptors.

## Built with
Static HTML, CSS, and JavaScript, hosted on GitHub Pages. Figures and animations
created in Blender, Molecular Nodes, 3D Slicer, and Python.

# Santiago Benavides — Biomedical Engineering Portfolio

**Live site → https://alma26x.github.io/**

A portfolio of independent biomedical-engineering and neuroscience work, spanning
neural decoding and brain imaging, medical-device design, tissue-engineering
scaffolds, molecular/structural visualization, and 3D biomedical animation.

## Sections
- **Neuroimaging & Computational Neuroscience** — MRI segmentation, DTI tractography, fMRI connectivity
- **Biomedical Devices** — BCI concept hardware, cardiac models, microneedle arrays, an ECG circuit
- **Tissue Engineering** — porous scaffold architectures
- **Molecular & Protein** — structural visualization and AI-assisted protein design
- **Animation & Visualization** — motor-unit and HD-EMG animations, a neuromuscular-junction render

## Built with
Static HTML / CSS / JavaScript, hosted on GitHub Pages. Figures and animations
created in Blender, Molecular Nodes, 3D Slicer, and Python.
